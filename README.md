# UXUY Skills Hub

Lightweight skill collection for UXUY-related agent workflows.

## Installation

Install this skills repo with:

```bash
npx skills add https://github.com/uxuyprotocol/skills-hub
```

## Included Skills

- `uxuy-dex`: query UXUY DEX market data through the public RPC gateway

## Notes

- Skill details live under `skills/`
- Each skill is defined by its own `SKILL.md`
